## Dependencies

- body-parser
- chance
- express
- mongodb
- md5
- moment
- nodemon
- node.js


1. Install dependencies using the `npm install` command.
2. Start the web server using the `npm run local` command. The app will be served at <http://localhost:8080/>.
3. Go to <http://localhost:8080/> in your browser.
